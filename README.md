# Deen Duel 

Deen Duel is a simple and engaging web-based quiz competition designed for two teams only.  
It is built for family gatherings, friends, and especially Ramadan occasions to promote religious and cultural awareness in a fun and friendly way.

---

##  Project Idea

Deen Duel is a two-team competitive quiz platform where users:

- Select a category
- Enter team names
- Answer timed questions
- Compete using a star-based scoring system
- View final scores on a leaderboard

The goal is to combine learning and entertainment in a clean and easy-to-use experience.

---

## Target Audience

- Families
- Friends
- Ramadan gatherings
- Casual religious competitions

---

## Core Features

- Smooth navigation between all pages
- Category selection before starting:
  - Cultural
  - Cooking
  - Fiqh
  
- Team 1 and Team 2 name input before starting
- **2-minute timer per question**
- Star-based scoring system 

- Total stars displayed at the top:
  - Right side: Team 1
  - Left side: Team 2
  
- Leaderboard page
- Modern and organized UI design
- Calm green and beige color palette
- Clean Arabic title typography
- Professional UI icons

- Exit confirmation alert:
  - “Are you sure you want to exit?”
  - Returns to homepage after confirmation
  
- Footer: © All Rights Reserved 2026
- Dedicated page: **“What is Deen?”**
  - Explains idea, purpose, benefits, and gameplay

## Prompt-Based Development (Written in Arabic)

This project was developed using structured prompts written in Arabic as part of a Vibe Coding workflow.

### Original Prompt (Arabic):
```
بصفتك خبير في هندسة البرمجيات بخبرة أكثر من 10 سنوات  
أنشئ موقع مسابقة تفاعلي بسيط باسم "دين" مخصص لتنافس فريقين فقط  
1. ربط جميع الصفحات ببعضها بشكل صحيح  
2. اختيار تصنيف قبل بدء الأسئلة (ثقافي – طبخ – فقه)  
3. إدخال اسم الفريق الأول واسم الفريق الثاني قبل البدء  
4. مدة كل سؤال دقيقتين فقط  
5. عرض النقاط على شكل نجوم تتنافس بين الفريقين  
6. لوحة متصدرين  
7. استخدم أيقونات احترافية لتصميم الموقع  
8. عند محاولة الخروج يظهر تنبيه: "هل أنت متأكد من الخروج؟" ثم العودة للصفحة الرئيسية عند التأكيد  
9. عرض اسم الموقع "دين" بخط عربي جميل في الأعلى  
10. إضافة تذييل: جميع الحقوق محفوظة 2026  
11. التصميم يكون نظيف، منظم، وسهل الاستخدام مع انتقالات سلسة بين الصفحات  
الجمهور المستهدف هم العائلة والأصدقاء  
ركز على المحتوى الديني والرمضاني والفائدة لنشر الوعي الديني والثقافي واستخدم نبره ودودة وبسيطة للفهم  
```
